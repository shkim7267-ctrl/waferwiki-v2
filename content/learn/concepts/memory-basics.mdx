메모리는 데이터를 저장하는 소자로 DRAM, NAND 등이 대표적입니다.
밀도, 속도, 전력, 내구성 사이에는 항상 트레이드오프가 존재합니다.
3D 구조와 공정 혁신이 메모리 성능을 좌우합니다.
메모리는 공정 혁신이 가장 빠르게 나타나는 영역 중 하나입니다.
